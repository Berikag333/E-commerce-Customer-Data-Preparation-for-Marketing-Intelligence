# E-commerce-Customer-Data-Preparation-for-Marketing-Intelligence
This project prepares raw e-commerce customer data for a Customer Loyalty Program by cleaning, standardizing, and validating customer profiles using Python. The resulting dataset is structured to support customer segmentation, KPI generation, and data-driven marketing decisions.

# E-commerce Customer Data Preparation for Marketing Intelligence

## Project Overview
This project supports the preparation of customer data for the launch of a **Customer Loyalty Program** at an e-commerce company (Store 1).  
The objective is to ensure that customer information is **clean, consistent, and ready for analysis**, enabling reliable insights and data-driven marketing decisions.

The work focuses on **data cleaning, standardisation, and validation**, creating a solid foundation for customer segmentation, KPI development, and future analytical work.

---

## Business Objective
Prepare customer data to:
- Enable personalised loyalty and retention campaigns
- Support customer segmentation by age, purchase behaviour, and product categories
- Ensure accurate and reliable marketing and business KPIs

---

## Dataset Description
The dataset is provided as a **Python list**, where each record represents a customer profile with the following fields:

| Column | Description |
|------|-------------|
| `usuario_id` | Unique customer identifier |
| `usuario_nombre` | Customer name (inconsistent formatting) |
| `usuario_edad` | Customer age |
| `categorias_fav_low` | Purchased product categories |
| `gasto_por_categoria` | Total spend per category |

---

## Work Performed
- Cleaned and standardised customer profiles  
- Normalised customer names and age values  
- Calculated total spend per customer  
- Validated data consistency and corrected structural issues  
- Prepared structured data for KPI calculation  
- Delivered an analysis-ready dataset for segmentation and reporting  

---

## Key Outputs
- Clean and standardised customer dataset  
- Customer-level total spending metrics  
- Verified alignment between product categories and spend data  
- Dataset ready for KPI development and advanced analytics
